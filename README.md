# Paradise Nursery Shopping Application

## Project Overview
Paradise Nursery is a shopping cart web application for an online plant store. 
Users can browse houseplants, add them to the cart, manage quantities, and view total costs.

## Features
- Display houseplants by categories
- Add plants to cart
- Increase and decrease quantity
- Remove items from cart
- Dynamic cart count
- Total cost calculation

## Technologies Used
- React
- Redux Toolkit
- React Router
- JavaScript
- CSS
